HeightMate Pro is a web-based application designed to track and visualize childrenâ€™s height growth over time. The app allows parents or caregivers to create individual kid profiles, record height measurements, and analyze growth patterns using interactive charts.

The application supports both manual height entry and Bluetooth-based automatic height input from compatible devices. Data is stored securely in the cloud using Firebase, making it accessible across devices. The frontend is built using HTML, CSS, and JavaScript and is hosted on GitHub Pages.

## Features
- Create and manage multiple kid profiles
- Record height, age, and date-wise measurements
- View growth statistics and interactive charts
- Bluetooth device integration for automatic height input
- Cloud-based data storage using Firebase
- Responsive and user-friendly interface

## Tech Stack
- HTML, CSS, JavaScript
- Firebase (Firestore)
- Chart.js
- Web Bluetooth API
- GitHub Pages (Hosting)
